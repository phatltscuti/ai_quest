Automate Video with Antigravity + Remotion

Bài viết này hướng dẫn bạn từng bước để:

  • Tự động tạo video bằng Antigravity + Remotion (dựa theo video Surya: AntiGravity's New VIDEO Editing (Remotion) Skills Are Insane).
  • Hiển thị chính video bạn tạo được trên một bài blog HTML.

1. Antigravity là gì? Remotion là gì?

Google Antigravity

Google Antigravity là một IDE (môi trường phát triển tích hợp) miễn phí của Google, được thiết kế cho lập trình viên và người dùng AI. Điểm đặc biệt của Antigravity là tích hợp sẵn AI agent (Gemini, Claude...) có khả năng nhận prompt bằng ngôn ngữ tự nhiên, tự lên kế hoạch và viết code cho bạn. Antigravity hỗ trợ Agent Skills – các bộ quy tắc chuyên biệt giúp agent hiểu rõ hơn về một lĩnh vực cụ thể (ví dụ: tạo video, thiết kế UI, viết test...).

Remotion

Remotion là một framework mã nguồn mở cho phép bạn tạo video bằng code (React + TypeScript). Thay vì kéo thả timeline trong phần mềm chỉnh sửa video truyền thống (Premiere, Final Cut...), bạn viết các React component để định nghĩa scene, animation, transition – và Remotion sẽ render thành file MP4. Remotion cung cấp skill remotion-best-practices để AI agent (trong Antigravity hoặc Claude Code) hiểu cách viết code video đúng chuẩn.

Kết hợp Antigravity + Remotion

Khi kết hợp hai công cụ này, bạn chỉ cần mô tả video bằng tiếng Anh trong Antigravity, agent sẽ tự viết code Remotion, và bạn nhận được video hoàn chỉnh – không cần biết chỉnh sửa video thủ công. Đây chính là workflow được giới thiệu trong video "AntiGravity's New VIDEO Editing (Remotion) Skills Are Insane" của Surya.

2. Tổng quan workflow

Quy trình tạo video tự động gồm 4 bước chính:

  1. Dùng Remotion làm engine video (React + TypeScript).
  2. Dùng Antigravity + skill remotion-best-practices để viết code video từ prompt tiếng Anh.
  3. Render video thành file MP4.
  4. Upload lên YouTube và nhúng vào bài blog.

3. Chuẩn bị môi trường

3.1. Cài công cụ cần thiết

  1. Cài Node.js LTS từ trang chính thức.
  2. Cài Google Antigravity IDE (Windows/macOS) từ https://antigravity.google.

3.2. Tạo project Remotion

Mở terminal (có thể dùng terminal tích hợp trong Antigravity) và chạy:

npx create-video@latest

Trong wizard của Remotion, chọn:

  • Template: Blank
  • TailwindCSS: Yes
  • Install Skills: Yes (để có Remotion Skills / remotion-best-practices)

3.3. Cài dependency và chạy dev server

cd <ten-project-remotion>
npm install
npm run dev

Mở trình duyệt tại http://localhost:3000 và kiểm tra Remotion đã hiển thị preview.

4. Kết nối Antigravity với Remotion skill

4.1. Mở project Remotion trong Antigravity

Sau khi chạy npx create-video@latest ở bước 3, Remotion sẽ tạo ra một thư mục project (ví dụ my-video). Bây giờ bạn cần mở thư mục đó trong Antigravity:

  1. Mở Google Antigravity.
  2. Vào menu File > Open Folder (hoặc nhấn Ctrl+K Ctrl+O trên Windows).
  3. Duyệt tới thư mục project Remotion vừa tạo (ví dụ C:\Users\ADMIN\my-video) và nhấn Select Folder.
  4. Antigravity sẽ mở workspace với cấu trúc thư mục của project Remotion (bạn sẽ thấy các file như package.json, src/, remotion.config.ts...).

4.2. Kiểm tra Remotion Skill đã cài chưa

Nếu ở bước 3.2 bạn đã chọn Install Skills: Yes, thì skill remotion-best-practices đã được cài sẵn trong project. Để kiểm tra:

  1. Trong Antigravity, mở Explorer (thanh bên trái), tìm thư mục .cursor/rules/ hoặc .antigravity/skills/ trong project.
  2. Nếu thấy file liên quan đến remotion-best-practices (hoặc remotion.mdc) là OK.
  3. Nếu không thấy, bạn có thể cài thủ công:
    • Vào trang remotion.dev/docs/skills và làm theo hướng dẫn.
    • Hoặc xem phần cài skill trong video của Surya (từ phút đầu).

4.3. Mở Agent và kiểm tra skill hoạt động

  1. Trong Antigravity, mở Agent panel (biểu tượng agent ở thanh bên trái, hoặc vào View > Agent).
  2. Gửi một prompt thử đơn giản để kiểm tra: Read the Remotion best-practices skill and summarize it.
  3. Nếu agent trả về được nội dung về các quy tắc Remotion (như cách tạo composition, cách dùng useCurrentFrame()...) thì skill đã sẵn sàng.
  4. Bây giờ bạn có thể chuyển sang bước 5 để bắt đầu tạo video bằng prompt.

5. Step-by-step: Tạo video bằng prompt trong Antigravity

5.1. Chuẩn bị asset (logo / hình ảnh)

Project Remotion đã có sẵn thư mục public/ (ví dụ my-video/public/). Đây là nơi chứa các file tĩnh (ảnh, font, audio...) mà video có thể sử dụng.

  1. Mở thư mục my-video/public trong Explorer của Antigravity.
  2. Copy logo hoặc ảnh bạn muốn dùng vào đây (ví dụ logo.png).
    Đường dẫn cuối cùng sẽ là: my-video/public/logo.png
  3. Nếu muốn gọn gàng hơn, bạn có thể tạo thêm thư mục con assets bên trong public/: 
    my-video/public/assets/logo.png
    Nhưng không bắt buộc – để thẳng trong public/ cũng được.

Lưu ý: Trong prompt ở bước 5.3, đường dẫn ảnh phải khớp với vị trí bạn đặt file. Ví dụ nếu để ở public/logo.png thì trong prompt ghi public/logo.png, nếu để ở public/assets/logo.png thì ghi public/assets/logo.png.

5.2. Mở agent và khai báo yêu cầu

  1. Trong Antigravity, mở một agent trong workspace Remotion.
  2. Chọn model phù hợp (Gemini/Claude tuỳ cấu hình) và sẵn sàng nhận prompt.

5.3. Prompt tạo video (ví dụ thực tế)

Dưới đây là prompt mẫu để tạo video giới thiệu cá nhân / portfolio:

Read the Remotion best-practices skill and create a 15-second personal introduction video.

Profile:
- Name: Ly Tan Phat
- Role: PHP Developer
- Company: Scuti
- Avatar/Logo: public/logo.png
- Tagline: "Building scalable web solutions with PHP, Laravel & modern tech"
- Skills highlight: PHP, Laravel, MySQL, REST API, Docker, Git

Requirements:
- Scene 1 (0-5s): Cinematic dark blue background, avatar image fades in center with a subtle glow effect, name "Ly Tan Phat" appears below with a smooth typewriter animation
- Scene 2 (5-12s): Transition to role & company info - "PHP Developer @ Scuti" with company-style layout, then skills icons/badges animate in one by one (PHP, Laravel, MySQL, REST API, Docker, Git)
- Scene 3 (12-18s): Tagline text reveals with elegant motion - "Building scalable web solutions with PHP, Laravel & modern tech", subtle particle or code-rain background effect
- Scene 4 (18-20s): Call-to-action - "Let's connect!" with GitHub/LinkedIn icons, smooth fade out

Style:
- Primary background: dark navy blue (#0a1628)
- Accent color: electric blue (#3b82f6) and white text
- Font: modern sans-serif (Inter or similar)
- Smooth cinematic transitions between scenes
- Use the avatar from public/logo.png

Export: 1920x1080, 30fps composition named "PhatDevIntro"

Sau khi gửi prompt:

  • Antigravity đọc rules trong skill Remotion.
  • Agent đề xuất một implementation plan (kế hoạch implement: tạo file nào, component nào, hiệu ứng gì).
  • Khi bạn chọn Proceed, agent sẽ viết code React/TypeScript Remotion cho video.

5.4. Xem preview và tinh chỉnh

  1. Quay lại tab Remotion (dev server đang chạy ở npm run dev).
  2. Chọn composition PhatDevIntro trong UI để xem preview.
  3. Nếu muốn tinh chỉnh:
    • Gửi thêm prompt trong Antigravity, ví dụ: Make the text appear with a typewriter effect.
    • Hoặc: Slow down the outro and add a subtle zoom on the logo.

5.5. Render video MP4 (tự động bằng code)

Remotion cho phép render video bằng CLI. Từ thư mục project, chạy:

npx remotion render PhatDevIntro out/phat-dev-intro.mp4

Sau khi lệnh hoàn thành, bạn sẽ có file out/phat-dev-intro.mp4 – chính là video được tạo từ workflow Antigravity + Remotion.
