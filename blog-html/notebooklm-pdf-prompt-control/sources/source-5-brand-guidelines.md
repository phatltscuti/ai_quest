# EduNova Brand Guidelines

**Version:** 3.0 | **Last Updated:** November 2025
**Brand Team Contact:** brand@edunova.vn

---

## 1. Brand Identity

### Mission
Democratize quality education through AI-powered technology.

### Vision
Every learner in Southeast Asia has access to a world-class AI tutor.

### Brand Values
- **Empowerment:** We believe education transforms lives
- **Innovation:** AI is our tool, not our product — learning is the product
- **Accessibility:** Technology should bridge gaps, not create them
- **Trust:** We earn trust through transparency, privacy, and accuracy

### Brand Voice
- **Confident** but not arrogant
- **Warm** but not informal
- **Smart** but not condescending
- **Forward-looking** but grounded in data

## 2. Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|-----------|----------|-----|-------|
| EduNova Blue (Primary) | #0d47a1 | 13, 71, 161 | Headlines, primary buttons, header bars, navigation |
| EduNova Orange (Accent) | #ff6d00 | 255, 109, 0 | CTAs, highlights, accent elements, data callouts |
| White | #ffffff | 255, 255, 255 | Backgrounds, text on dark surfaces |
| Light Gray | #f5f5f5 | 245, 245, 245 | Card backgrounds, secondary surfaces |

### Secondary Colors

| Color Name | Hex Code | RGB | Usage |
|-----------|----------|-----|-------|
| Dark Text | #212121 | 33, 33, 33 | Body text, headings on light backgrounds |
| Secondary Text | #616161 | 97, 97, 97 | Captions, labels, supporting text |
| Success Green | #2e7d32 | 46, 125, 50 | Positive metrics, completion status |
| Warning Yellow | #f57f17 | 245, 127, 23 | Alerts, attention-needed indicators |
| Error Red | #c62828 | 198, 40, 40 | Errors, critical alerts, risk indicators |
| Border Gray | #e0e0e0 | 224, 224, 224 | Dividers, card borders |

### Color Usage Rules
- **DO:** Use EduNova Blue as the dominant color (60-70% of colored elements)
- **DO:** Use Orange sparingly for emphasis (10-20%)
- **DO:** Maintain high contrast (minimum 4.5:1 ratio for text)
- **DON'T:** Use gradients on official materials
- **DON'T:** Introduce new colors outside this palette
- **DON'T:** Place orange text on blue backgrounds (low contrast)

## 3. Typography

### Primary Typeface: Inter

| Style | Weight | Size | Usage |
|-------|--------|------|-------|
| Display | Bold (700) | 36-48px | Hero headings, slide titles |
| Heading 1 | Bold (700) | 28-32px | Section headings |
| Heading 2 | Semi-Bold (600) | 22-24px | Subsection headings |
| Heading 3 | Semi-Bold (600) | 18-20px | Card titles, labels |
| Body | Regular (400) | 16px | Paragraph text, descriptions |
| Caption | Regular (400) | 13-14px | Footnotes, labels, metadata |

### Secondary Typeface: Roboto Mono
- Used only for: code, data tables, technical specifications
- Weight: Regular (400), size: 14px

### Typography Rules
- **Line height:** 1.5x for body text, 1.2x for headings
- **Letter spacing:** Normal for body, -0.5px for display headings
- **Maximum line length:** 70 characters for body text
- **DON'T:** Use more than 3 font weights on a single page/slide
- **DON'T:** Use italic for emphasis — use bold or color instead

## 4. Logo Usage

### Logo Variants
1. **Full logo:** EduNova icon + wordmark (preferred for most uses)
2. **Icon only:** Square icon (for app icons, favicons, small spaces)
3. **Wordmark only:** Text-only "EduNova" (for inline text, formal documents)

### Clear Space
- Minimum clear space around logo = height of the "E" in EduNova on all sides
- No text, images, or other elements within this zone

### Logo Colors
- **On white/light backgrounds:** Blue icon + dark text wordmark
- **On dark/blue backgrounds:** White icon + white wordmark
- **On images:** Use white version with semi-transparent dark overlay behind logo

### Logo Don'ts
- DON'T rotate, skew, or distort the logo
- DON'T change logo colors outside approved variants
- DON'T place logo on busy/cluttered backgrounds
- DON'T resize below minimum size (24px height for digital, 10mm for print)

## 5. Presentation / Slide Design

### Slide Layout Principles
- **Left-aligned text** with generous margins (minimum 10% padding on all sides)
- **One key message per slide** — if it takes more than 5 seconds to understand, simplify
- **60/40 rule:** 60% visual elements (charts, icons, images), 40% text
- **Maximum 40 words** of body text per slide
- **Maximum 5 bullet points** per slide, each under 12 words

### Slide Structure
- **Header bar:** Thin dark-blue (#0d47a1) stripe at top of every slide with white section label
- **Title:** Large, bold, dark blue text below header
- **Content area:** Left-aligned text + right-aligned visual (or full-width visual)
- **Footer:** Slide number (bottom right), "EduNova Confidential" (bottom left) in light gray

### Data Visualization
- **Preferred chart types:** Bar charts, donut charts, stat cards, horizontal timelines
- **Chart colors:** Use primary palette only (blue, orange, gray). Maximum 4 colors per chart
- **Labels:** Always include data labels directly on charts (no legends if possible)
- **Icons:** Flat, single-color, modern style. Only blue or orange fill.
- **DON'T:** Use 3D charts, pie charts with more than 4 segments, or decorative chart elements

### Imagery
- **Preferred:** Abstract geometric illustrations, flat-design technology graphics
- **Acceptable:** Product screenshots, team photos (for culture slides only)
- **DON'T:** Use generic stock photos, clip art, or images with watermarks
- **DON'T:** Use images of students/children without proper consent documentation

## 6. Iconography

### Style
- **Flat, single-color icons** (no gradients, no shadows, no outlines)
- **Color:** EduNova Blue (#0d47a1) or Orange (#ff6d00) on light backgrounds; White on dark backgrounds
- **Size:** 24px (inline), 32px (navigation), 48px (feature highlights), 64px+ (hero sections)
- **Source:** Material Design Icons or Phosphor Icons (consistent set per project)

### Icon Don'ts
- DON'T mix icon styles (e.g., outlined + filled in same context)
- DON'T use more than 2 colors per icon
- DON'T use icons smaller than 16px

## 7. Writing Style

### Tone by Context
| Context | Tone | Example |
|---------|------|---------|
| Marketing / Website | Warm, inspiring | "Transform your teaching with AI" |
| Product / UI | Clear, helpful | "Your lesson plan is ready" |
| Investor / Board | Confident, data-driven | "Revenue grew 57.7% to $28.7M" |
| Support / Help | Patient, friendly | "Let's get this sorted out" |

### Writing Rules
- Use active voice: "EduNova serves 2.1M users" (not "2.1M users are served by EduNova")
- Lead with numbers: "$28.7M revenue" (not "Revenue of twenty-eight point seven million")
- Avoid jargon with external audiences: "AI system that personalizes learning" (not "fine-tuned LLM with RAG pipeline")
- Use "we" for company, "you" for customer/user
- Sentence case for headings (not Title Case or ALL CAPS)
